UlyssesCode

Code for questions from an autistic kid's curious mind

These programs were created to answer the persistent questions of Ulysses, a brilliant kid on the autism spectrum who loves patterns and math. When he gets obsessed with something, he demands more and more examples. These tools help his teachers, therapists, and family keep up with his endless curiosity.

Color Mixing

Ulysses was obsessed with what color is produced when two colors are mixed. He memorized primary, secondary, and tertiary colors, but more complicated mixtures became challenging. This program translates the Red-Yellow-Blue (RYB) subtractive color model (used in paint mixing) into the Red-Green-Blue (RGB) additive model used by computers, allowing complex color combinations to be calculated and displayed.

Features:

  • RYB to RGB color conversion
  • Support for white and black to adjust brightness
  • Comprehensive color name lookup table
  • HEX color code output

Counting Programs

Since age 4, counting has been one of Ulysses's most persistent interests. He counts by intervals, in different bases, and by patterns he creates. These Python scripts, designed for his TI-84 CE calculator, answer his repeated "what comes after?" questions.

Available Programs:

CNT_BASE Count in any base
CNT_BAS1 Base counting variant
CNT_N_BS Count in N bases
CNT_DUBL Doubling sequences
CNT_FLIP Flip counting patterns
CNT_POWR Power sequences
CNT_SKIP Skip counting
CNT_0REP Zero repetition patterns
CNT_1REP One repetition patterns
SKIP_NUM Skip number sequences
SKIP_TYP Skip type patterns
NUM_TYPE Number type identification
